## 📘 Descripción

Este repositorio contiene un **curso interactivo sobre Complejidad Computacional**, desarrollado como material didáctico para la asignatura. La página web incluye explicaciones teóricas, visualizaciones dinámicas, ejemplos paso a paso y quizzes interactivos para reforzar el aprendizaje.

🌐 **Sitio web:** [miriamfarfan.github.io/ESTADISTICA_Y_ML/complejidad_computacional](https://miriamfarfan.github.io/ESTADISTICA_Y_ML/complejidad_computacional)

---

## 📚 Contenido del Curso

El material está organizado en **5 bloques temáticos**:

| Bloque | Título | Descripción |
|--------|--------|-------------|
| 1 | **Inducción Matemática** | Base, hipótesis inductiva, paso inductivo. Ejemplos con sumas, factorial y Fibonacci. |
| 2 | **Notación Asintótica** | Big-O, Ω, Θ, o, ω. Comparación de funciones y gráficas interactivas. |
| 3 | **Representación de Grafos** | Matriz de adyacencia, lista de adyacencia, lista de aristas. Constructor visual de grafos. |
| 4 | **Análisis con Big-O** | Reglas de conteo, pseudocódigos animados, complejidad de bucles y recursión. |
| 5 | **Diseño de Algoritmos** | Estrategias: fuerza bruta, dos punteros, conteo. Problemas resueltos paso a paso. |

---

## ✨ Características Interactivas

| Característica | Descripción |
|----------------|-------------|
| 🎮 **Constructor de grafos** | Crea nodos y aristas haciendo clic, visualiza matrices y listas de adyacencia en tiempo real. |
| 📈 **Gráficas dinámicas** | Compara funciones de complejidad (O(1), O(log n), O(n), O(n²), O(2ⁿ)) con Chart.js. |
| 🎬 **Pseudocódigos animados** | Ejecuta paso a paso algoritmos y observa su complejidad. |
| ✏️ **Quizzes interactivos** | 5 quizzes (uno por bloque) con retroalimentación inmediata y puntuación. |
| 🏆 **Sistema de progreso** | Barra de progreso global y puntuación acumulada por bloque. |
| 🔍 **Visualización de inducción** | Tablas que verifican fórmulas para diferentes valores de n. |

## 🚀 Cómo Usar

1. **Accede al sitio web:** [https://miriamfarfan.github.io/ESTADISTICA_Y_ML/complejidad_computacional/](https://miriamfarfan.github.io/ESTADISTICA_Y_ML/complejidad_computacional/)
2. Navega por los bloques usando el menú lateral

## 🎯 Objetivos de Aprendizaje

Al completar este curso, los estudiantes serán capaces de:

- ✅ Demostrar propiedades mediante **inducción matemática**
- ✅ Comparar funciones usando **notación asintótica** (O, Ω, Θ)
- ✅ Representar **grafos** en diferentes estructuras de datos
- ✅ Analizar la **complejidad temporal** de algoritmos
- ✅ Diseñar **algoritmos simples** para problemas básicos
